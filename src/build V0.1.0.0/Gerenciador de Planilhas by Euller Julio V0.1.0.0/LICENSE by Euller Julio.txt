## Descrição do Aplicativo

Este aplicativo é um gerenciador financeiro desenvolvido para facilitar a organização e o controle de dados financeiros. Ele permite a inserção e o armazenamento de informações financeiras, como datas, valores, fornecedores e descrições, em uma planilha Excel. O aplicativo automatiza o processo de criação de abas para diferentes centros de custo, permitindo uma visualização mais clara e organizada dos dados financeiros.
